Class LiteStorage<T>

轻量的本地文件持久性数据存储。主要用于简单的配置信息持久化

Example

const stor = await new LiteStorage({ uuid: 'test1' }).ready();
const config = stor.get();
console.log(config);
stor.getItem('key1');
stor.setItem('key2', { a: 1 });
stor.removeItem('key2');
stor.clear();
